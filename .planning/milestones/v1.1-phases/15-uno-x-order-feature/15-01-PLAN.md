  <files>
    src/db/schema/bonus-riders.ts

  <action>
    Create `src/db/schema/bonus-riders.ts` with a `bonusRiders` table:
    - `id`: serial primary key
    - `leagueId`: integer NOT NULL, references leagues.id with onDelete cascade
    - `teamId`: integer NOT NULL, references teams.id
    - `riderId`: integer NOT NULL, references riders.id
    - `raceId`: integer NOT NULL, references races.id (this is the parent GT race, NOT a stage)
    - `orderId`: integer, references orders.id (nullable - links back to the Uno-X order that triggered this)
    - `pickOrder`: integer NOT NULL (1-based, indicating when this team picked in the reverse-standings draft)
    - `pickedAt`: timestamp with timezone, NOT NULL, default now()

    Add indexes:
    - `bonus_riders_league_idx` on leagueId
    - `bonus_riders_team_idx` on teamId
    - `bonus_riders_race_idx` on raceId
    - `bonus_riders_league_race_team_unique` uniqueIndex on (leagueId, raceId, teamId) — one bonus rider per team per GT

    Add relations:
    - league -> leagues
    - team -> teams
    - rider -> riders
    - race -> races
    - order -> orders (optional)

    Export from `src/db/schema/index.ts` by adding `export * from "./bonus-riders"`.

    Create SQL migration at `src/db/migrations/0002_add_bonus_riders.sql`:
    ```sql
    CREATE TABLE IF NOT EXISTS "bonus_riders" (
      "id" serial PRIMARY KEY,
      "leagueId" integer NOT NULL REFERENCES "leagues"("id") ON DELETE CASCADE,
      "teamId" integer NOT NULL REFERENCES "teams"("id"),
      "riderId" integer NOT NULL REFERENCES "riders"("id"),
      "raceId" integer NOT NULL REFERENCES "races"("id"),
      "orderId" integer REFERENCES "orders"("id"),
      "pickOrder" integer NOT NULL,
      "pickedAt" timestamp with time zone NOT NULL DEFAULT now()
    );
    CREATE INDEX IF NOT EXISTS "bonus_riders_league_idx" ON "bonus_riders" ("leagueId");
    CREATE INDEX IF NOT EXISTS "bonus_riders_team_idx" ON "bonus_riders" ("teamId");
    CREATE INDEX IF NOT EXISTS "bonus_riders_race_idx" ON "bonus_riders" ("raceId");
    CREATE UNIQUE INDEX IF NOT EXISTS "bonus_riders_league_race_team_unique" ON "bonus_riders" ("leagueId", "raceId", "teamId");
    ```

    Follow the existing schema patterns from orders.ts and draft.ts (pgTable, relations, indexes).
