  <action>
Create `src/lib/scoring-queries.ts` with the following exported functions:

**1. `getLeagueStandings(leagueId: number, season: number)`**
- Returns array of `{ teamId, teamName, userId, totalPoints, rank }` sorted by totalPoints DESC
- Start from `teams` table filtered by `leagueId`
- LEFT JOIN `draftPicks` with `eq(draftPicks.teamId, teams.id)` AND `eq(draftPicks.leagueId, leagueId)`
- LEFT JOIN `raceResults` with `eq(raceResults.riderId, draftPicks.riderId)`
- LEFT JOIN `races` with `eq(races.id, raceResults.raceId)` AND `eq(races.season, season)` — this season-scopes the results
- Use `sql<number>\`COALESCE(SUM(${raceResults.points}), 0)\`` for totalPoints
- `.groupBy(teams.id, teams.name, teams.userId)`
- Sort by the raw SUM expression DESC (not the alias — use `desc(sql\`COALESCE(SUM(${raceResults.points}), 0)\`)`) to avoid PostgreSQL alias resolution issues
- Derive rank from sorted array index in JS: `rows.map((row, i) => ({ ...row, rank: i + 1 }))`. Handle ties: if `rows[i].totalPoints === rows[i-1].totalPoints`, use `rows[i-1].rank`

**2. `getTeamRiderScores(teamId: number, leagueId: number, season: number)`**
- Returns array of `{ riderId, riderName, riderTeam, gender, totalPoints }` sorted by totalPoints DESC
- Start from `draftPicks` filtered by `eq(draftPicks.teamId, teamId)` AND `eq(draftPicks.leagueId, leagueId)`
- INNER JOIN `riders` on `eq(riders.id, draftPicks.riderId)`
- LEFT JOIN `raceResults` on `eq(raceResults.riderId, draftPicks.riderId)`
- LEFT JOIN `races` on `eq(races.id, raceResults.raceId)` AND `eq(races.season, season)`
- Use `sql<number>\`COALESCE(SUM(${raceResults.points}), 0)\`` for totalPoints
- `.groupBy(draftPicks.riderId, riders.name, riders.team, riders.gender)`
- Sort by SUM expression DESC

**Import pattern:** Follow existing `draft-queries.ts` style. Import from `@/lib/db`, individual schema files (not `@/db/schema` barrel), and `drizzle-orm` operators.

**Type exports:** Export `LeagueStanding` and `TeamRiderScore` types derived from the return types.
