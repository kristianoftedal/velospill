**Query 1: Team metadata + league name**
- SELECT from teams JOIN leagues ON leagues.id = teams.leagueId
- WHERE teams.id = teamId AND teams.leagueId = leagueId
- Returns: teamId, teamName, leagueId, leagueName, (optional: totalPoints via SUM)
- If no row returned → return null (team not found or wrong league)

**Query 2: Per-rider, per-race scoring breakdown (main roster)**
- Query draftPicks JOIN riders JOIN raceResults JOIN races
- Filter: draftPicks.teamId = teamId AND draftPicks.leagueId = leagueId
- Filter: races.season = season AND races.startDate >= draftPicks.pickedAt (ownership-at-race-time)
- Filter: lineupFilter (only riders in lineup score, or all if no lineup submitted)
- Filter: league race scoping — sql`(${races.id} IN (SELECT "raceId" FROM league_races WHERE "leagueId" = ${leagueId}) OR ${races.parentRaceId} IN (SELECT "raceId" FROM league_races WHERE "leagueId" = ${leagueId}))`
- SELECT: riderId, riderName, riderTeam, gender, raceId, raceName, raceType, startDate, raceResults.points, raceResults.category
- ORDER BY: riders.name ASC, races.startDate ASC
- Group in application code: outer Map keyed by riderId → inner Map keyed by raceId → list of category scores per race

**Query 3: Bonus riders for this team**
- Query bonusRiders JOIN riders JOIN raceResults JOIN races
- Filter: bonusRiders.teamId = teamId AND bonusRiders.leagueId = leagueId
- Filter: races.season = season AND (races.id = bonusRiders.raceId OR races.parentRaceId = bonusRiders.raceId)
- Filter: league race scoping (same sql fragment as above)
- SELECT: riderId, riderName, riderTeam, gender, raceId, raceName, raceType, startDate, raceResults.points, raceResults.category, isBonus = true
- Merge bonus riders into the same rider map (keyed by riderId); mark isBonus: true

**Application-side assembly:**
Build TeamRiderEntry[] from the merged map:
- For each rider: riderId, riderName, riderTeam, gender, totalPoints (sum across all races), isBonus, races: TeamRiderRaceEntry[]
- For each race entry: raceId, raceName, raceType, startDate, racePoints (sum of category points for this rider/race), categories: { category, categoryLabel, position, points }[]
- Use the same categoryLabels map as rider-queries.ts
- Sort riders by totalPoints DESC (highest contributor first)
- Sort races within each rider by startDate ASC
