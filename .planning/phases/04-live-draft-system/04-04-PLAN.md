1. Start dev server: `npm run dev`
2. Open two browser windows (different users/sessions if possible, or use incognito)
3. Create a league, have both users join (or use the invite code flow)
4. As league owner: transition league to "Drafting" status
5. Navigate to `/leagues/[leagueId]/draft` in both windows
6. Verify: Both users see the waiting room with team list

10. Verify: Pick appears on the draft board in both windows instantly
11. Verify: Timer resets for the next drafter

14. After all picks: verify draft recap shows each team's full roster
