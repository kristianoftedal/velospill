       Step A — Get completed parent races for this league ordered by startDate ASC:
       Query `races` where `parentRaceId IS NULL`, season matches, race is in league_races for this leagueId, and at least one result exists (INNER JOIN raceResults ON raceResults.raceId = races.id filtered via draftPicks for leagueId). Group by race, order by startDate ASC.
       Use `isNull(races.parentRaceId)` from drizzle-orm for the null check.

       Step B — Get per-team per-parent-race points using the same ownership-at-race-time pattern as `getLeagueStandings`. For stage races, results on stages (where races.parentRaceId = parentRaceId) must roll up to the parent race column. Use COALESCE(races.parentRaceId, races.id) to bucket stage results under their parent race:
       ```sql
       SELECT teams.id, teams.name, teams.userId,
              COALESCE(r.parentRaceId, r.id) as parentRaceId,
              SUM(CASE WHEN lineupFilter THEN raceResults.points ELSE 0 END) as racePoints
       FROM teams
       LEFT JOIN draftPicks ON (draftPicks.teamId = teams.id AND draftPicks.leagueId = leagueId)
       LEFT JOIN raceResults ON raceResults.riderId = draftPicks.riderId
       LEFT JOIN races r ON (r.id = raceResults.raceId AND r.season = season AND r.startDate >= draftPicks.pickedAt AND league-scoping subquery)
       WHERE teams.leagueId = leagueId
       GROUP BY teams.id, teams.name, teams.userId, COALESCE(r.parentRaceId, r.id)
       ```
       In Drizzle: use `sql<number>\`COALESCE(${races.parentRaceId}, ${races.id})\`` as the groupBy/select key.

       Step C — Get bonus rider points per parent race (mirror the bonus query from getLeagueStandings, but also GROUP BY bonusRiders.raceId to get per-race values):
       ```sql
       SELECT bonusRiders.teamId, bonusRiders.raceId as parentRaceId,
              SUM(raceResults.points) as bonusPoints
       FROM bonus_riders
       INNER JOIN raceResults ON raceResults.riderId = bonusRiders.riderId
       INNER JOIN races ON races.id = raceResults.raceId
       WHERE bonusRiders.leagueId = leagueId AND races.season = season
         AND (races.id = bonusRiders.raceId OR races.parentRaceId = bonusRiders.raceId)
         AND league-scoping subquery
       GROUP BY bonusRiders.teamId, bonusRiders.raceId
       ```

       - totalPoints = last cumulative value (or sum of all race points)
       - Sort teams by totalPoints DESC
