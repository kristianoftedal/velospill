1. Create `src/app/(main)/leagues/[leagueId]/draft/page.tsx` (Server Component):
   - Authenticate user via `auth.api.getSession({ headers: await headers() })`; redirect to `/login` if no session
   - Parse `leagueId` from params (await params, parseInt)
   - Check league membership via `checkLeagueMembership`; if not member, redirect to `/leagues`
   - Load draft state via `getDraftState(leagueId)` and available riders via `getAvailableRiders(leagueId, 'M')` and `getAvailableRiders(leagueId, 'F')`
   - Get user's team from membership check
   - If no draft session exists (status still 'setup' or 'drafting' pre-start):
     - Show a waiting room UI: league name, "Waiting for draft to start..." message, list of teams who've joined
     - If user is league owner, show a "Start Draft" button that calls `startDraft` action
   - If draft session exists: render `<DraftRoom>` passing all loaded state as props:
     - `leagueId`, `initialSession`, `initialPicks`, `teams`, `availableMen`, `availableWomen`, `currentUserId`, `currentTeamId` (from user's team), `isOwner`

2. Create `src/app/(main)/leagues/[leagueId]/draft/draft-room.tsx` ("use client"):
   - Props: leagueId, initialSession, initialPicks, teams, availableMen, availableWomen, currentUserId, currentTeamId, isOwner
   - State management:
     - `picks` (array of draft picks, initialized from initialPicks)
     - `currentTurn` (teamId of whose turn it is)
     - `timerExpiresAt` (Date or null)
     - `draftStatus` ('pending' | 'men' | 'women' | 'complete')
     - `currentPickIndex` (number)
     - `availableRiders` (filtered list based on current gender)
   - Pusher subscription in useEffect:
     - Subscribe to `presence-draft-${leagueId}`
     - Bind events:
       - `'draft-started'`: set status, currentTurn, timer, etc.
       - `'pick-made'`: append to picks, update currentTurn, timerExpiresAt, update available riders (remove picked rider), if nextTeamId === currentTeamId play sound + show visual "YOUR TURN!" notification
       - `'draft-complete'`: set status to 'complete', clear timer
     - Cleanup: unsubscribe on unmount
   - "Your turn" notification:
     - Import `useSound` from `use-sound` for audio: `const [playYourTurn] = useSound('/sounds/your-turn.mp3')`
     - Visual: When it's user's turn, show a pulsing banner at top "It's your turn to pick!"
     - Place a simple MP3 file reference at `/public/sounds/your-turn.mp3` â€” use a placeholder comment noting the user should place an audio file there (Claude cannot create audio files)
   - Determine `isMyTurn`: `currentTurn === currentTeamId && draftStatus !== 'complete'`
   - Layout:
     - Top: Draft status banner (Men's Draft Round X / Women's Draft Round Y / Draft Complete), timer, current drafter name
     - Left/main: `<DraftBoard>` with picks grid
     - Right/bottom: `<RiderPicker>` (only interactive if `isMyTurn`)
   - Use Tailwind for responsive layout: on lg+ side-by-side (board left, picker right), on mobile stacked

3. Create `src/app/(main)/leagues/[leagueId]/draft/draft-board.tsx` ("use client"):
   - Props: teams, picks, currentPickIndex, draftStatus, menRounds (18), womenRounds (6)
   - Build the grid using `buildDraftOrder(teams, 18, 6)` to know which team goes in each cell
   - Render a table/grid with:
     - Column headers: team names
     - Row headers: "M Round 1", "M Round 2", ..., "M Round 18", "W Round 1", ..., "W Round 6"
     - Each cell: if pick exists for that slot, show rider name (with muted text for auto-picks). If it's the current pick slot, highlight it. Otherwise empty.
   - Horizontal scroll on mobile (`overflow-x-auto`)
   - Use `React.memo` on individual cells to avoid re-rendering all 240 cells on each pick
   - Styling: use shadcn Table or a custom Tailwind grid. Current pick cell has a colored border/highlight. Filled cells show rider name + rider team. Auto-picks shown in italics.
   - Section divider between men's and women's rounds

   - On pick click: call `onPick(riderId)` which triggers the parent's makePick action
   - Styling: Card container with header showing current gender draft label ("Men's Riders" / "Women's Riders")

2. Create `src/app/(main)/leagues/[leagueId]/draft/timer.tsx` ("use client"):
   - Props: `expiresAt` (Date | null), `isMyTurn` (boolean)
   - State: `secondsLeft` (number)
   - useEffect with setInterval (500ms polling):
     - Compute `Math.max(0, Math.floor((expiresAt.getTime() - Date.now()) / 1000))`
     - Update `secondsLeft`
     - Clear interval on unmount or when expiresAt changes
   - If `expiresAt` is null, render nothing (draft complete or not started)
   - Display:
     - Large timer number (text-4xl font-bold)
     - When `secondsLeft <= 10`: red text with `animate-pulse` class
     - When `secondsLeft <= 5`: add more urgency (larger text, red background)
     - When `isMyTurn`: show "Your time" label above timer
     - When not your turn: show "Time remaining" label
   - Format: show just seconds (e.g., "45s") since picks are 60s max

3. Wire RiderPicker and Timer into DraftRoom:

   - Pass `timerExpiresAt` to `<DraftTimer>`
   - Pass filtered riders (based on current gender) to `<RiderPicker>`
   - When Pusher `pick-made` event arrives, remove the picked rider from `availableRiders` state
