Implement `dropRider(data: { leagueId: number; riderId: number }): Promise<{ success: true } | { success: false; error: string }>` with these steps in order:

1. Auth: `getAuthenticatedUser()` from `@/lib/league-auth` — catch and return `{ success: false, error: "Unauthorized" }`.
2. Membership: `checkLeagueMembership(session.user.id, data.leagueId)` — if `!isMember || !team` return `{ success: false, error: "Not a member of this league" }`.
3. League active guard: fetch the league record and verify `league.status === "active"` — return `{ success: false, error: "Roster changes are only allowed in active leagues" }` if not. Use `db.select({ status: leagues.status }).from(leagues).where(eq(leagues.id, data.leagueId)).limit(1)`.
4. Ownership check: `db.select({ id: draftPicks.id }).from(draftPicks).where(and(eq(draftPicks.teamId, team.id), eq(draftPicks.leagueId, data.leagueId), eq(draftPicks.riderId, data.riderId))).limit(1)` — if no row return `{ success: false, error: "Rider is not on your team" }`.
5. Delete draftPicks row: `db.delete(draftPicks).where(and(eq(draftPicks.teamId, team.id), eq(draftPicks.leagueId, data.leagueId), eq(draftPicks.riderId, data.riderId)))`.
6. Cleanup IR: `db.delete(irRequests).where(and(eq(irRequests.teamId, team.id), eq(irRequests.leagueId, data.leagueId), eq(irRequests.riderId, data.riderId), inArray(irRequests.status, ["pending", "approved"])))`. Hard-delete — no cancelled status exists.
7. Cleanup transfer bids: `db.update(transferBids).set({ status: "cancelled" }).where(and(eq(transferBids.teamId, team.id), eq(transferBids.leagueId, data.leagueId), eq(transferBids.outRiderId, data.riderId), eq(transferBids.status, "pending")))`.
8. Revalidate: `revalidatePath("/leagues/${data.leagueId}/roster")` and `revalidatePath("/leagues/${data.leagueId}")` and `revalidatePath("/leagues/${data.leagueId}/ir")`.
9. Return `{ success: true }`.

Imports needed: `db` from `@/lib/db`, `irRequests` from `@/db/schema/ir`, `draftPicks` from `@/db/schema/draft`, `transferBids` from `@/db/schema/transfers`, `leagues` from `@/db/schema/leagues`, `eq, and, inArray` from `drizzle-orm`, `revalidatePath` from `next/cache`, `getAuthenticatedUser, checkLeagueMembership` from `@/lib/league-auth`.
