**Add to `scoring-queries.ts`:**

**1. `getRaceScoreBreakdown(raceId: number, leagueId: number)`**
- Returns array of `{ teamId, teamName, riderId, riderName, position, points }` sorted by position ASC
- Start from `raceResults` filtered by `eq(raceResults.raceId, raceId)`
- INNER JOIN `riders` on `eq(raceResults.riderId, riders.id)`
- INNER JOIN `draftPicks` on `eq(draftPicks.riderId, raceResults.riderId)` AND `eq(draftPicks.leagueId, leagueId)` â€” only show drafted riders
- INNER JOIN `teams` on `eq(teams.id, draftPicks.teamId)`
- Order by `raceResults.position` ASC
- Export type `RaceScoreEntry`

**2. `getLeagueRacesWithScores(leagueId: number, season: number)`**
- Returns array of `{ raceId, raceName, raceType, startDate, totalLeaguePoints }` for races where at least one drafted rider has results
- Start from `races` filtered by `eq(races.season, season)`
- INNER JOIN `raceResults` on `eq(raceResults.raceId, races.id)`
- INNER JOIN `draftPicks` on `eq(draftPicks.riderId, raceResults.riderId)` AND `eq(draftPicks.leagueId, leagueId)`
- Use `sql<number>\`COALESCE(SUM(${raceResults.points}), 0)\`` for totalLeaguePoints
- `.groupBy(races.id, races.name, races.raceType, races.startDate)`
- Order by `races.startDate` DESC (most recent first)
- Export type `LeagueRaceScore`
