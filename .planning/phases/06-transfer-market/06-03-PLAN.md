  <name>Task 1: Create admin transfer server actions</name>
  <files>src/app/admin/transfers/actions.ts</files>
  <action>
Create `src/app/admin/transfers/actions.ts` with "use server" directive. Follow the pattern from `admin/results/actions.ts` for `checkAdminAuth()`.

**getPendingBids():**
Query `transferBids` with status "pending", joined with `leagues` (name), `teams` (name), and `riders` twice using aliases (`outRider` and `inRider`) for rider names. Include `submittedAt`. Order by `submittedAt` ASC (oldest first for fairness). Return array of bid objects with all display data.

Use `alias(riders, "outRider")` and `alias(riders, "inRider")` from `drizzle-orm`.

**getBidHistory(limit = 50):**
Query `transferBids` with status NOT "pending" (approved, rejected, cancelled), same joins as above. Include `resolvedAt`, `adminNote`, `resolvedBy`. Order by `resolvedAt` DESC. Limit results.

**approveBid(bidId: number):**
This is the critical mutation. Must be fully transactional:
1. `checkAdminAuth()` — get session
2. Open `db.transaction(async (tx) => { ... })`
3. Fetch bid: `tx.query.transferBids.findFirst({ where: and(eq(id, bidId), eq(status, "pending")) })`. If not found → throw "Bid not found or not pending"
4. **Re-verify inRider is still a free agent** (race condition protection — per research Pitfall 1): Query `tx.query.draftPicks.findFirst({ where: and(eq(leagueId, bid.leagueId), eq(riderId, bid.inRiderId)) })`. If exists → throw "Incoming rider is no longer a free agent"
5. **Re-verify outRider still belongs to team:** Query `draftPicks` for `leagueId + teamId + outRiderId`. If not found → throw "Outgoing rider is no longer on this team"
6. **Get inRider gender** from `riders` table (needed for new `draftPick.gender`)
7. **Delete old draftPick:** `tx.delete(draftPicks).where(eq(draftPicks.id, currentPick.id))`
8. **Insert new draftPick:** Use `pickNumber: -(bidId)` as sentinel (negative = transfer, unique per bid). Set `round: -1`, `gender: inRider.gender`, `wasAutomatic: false`, `pickedAt: new Date()` — this `pickedAt` is critical for ownership-at-race-time scoring (Plan 01)
9. **Update bid status:** Set `status: "approved"`, `resolvedAt: new Date()`, `resolvedBy: session.user.id`
10. **Insert audit entry:** Action "APPROVED", performedBy session.user.id
11. End transaction
12. `revalidatePath("/admin/transfers")`
13. `revalidatePath(`/leagues/${bid.leagueId}/transfers`)`
14. `revalidatePath(`/leagues/${bid.leagueId}/standings`)` — standings affected by roster change
15. Return `{ success: true }`

Catch errors: if `error.code === "23505"` (unique constraint violation on `riderLeagueUnique`), return `{ success: false, error: "Rider is already on a team in this league" }`. Otherwise return generic error.

**rejectBid(bidId: number, adminNote: string):**
1. `checkAdminAuth()`
2. Fetch bid, verify status is "pending"
3. Update bid: `status: "rejected"`, `resolvedAt: new Date()`, `resolvedBy: session.user.id`, `adminNote`
4. Insert audit entry: action "REJECTED", note = adminNote
5. `revalidatePath` both admin and league transfers paths
6. Return `{ success: true }`
