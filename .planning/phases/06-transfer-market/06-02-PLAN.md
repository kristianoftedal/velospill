Create `src/lib/transfer-queries.ts` with these query functions:

**getFreeAgents(leagueId: number, gender: "M" | "F"):**
Returns all riders NOT currently on any team in this league, filtered by gender. Use a `notInArray` subquery against `draftPicks` for the given `leagueId`:
```typescript
const ownedRiderIds = db
  .select({ riderId: draftPicks.riderId })
  .from(draftPicks)
  .where(eq(draftPicks.leagueId, leagueId))

return db.select({
  id: riders.id,
  name: riders.name,
  team: riders.team,
  nationality: riders.nationality,
  gender: riders.gender,
}).from(riders)
  .where(and(
    notInArray(riders.id, ownedRiderIds),
    eq(riders.gender, gender)
  ))
  .orderBy(riders.name)
```

**getTeamRoster(teamId: number, leagueId: number):**
Returns all riders on a team via `draftPicks`, joined with `riders` for name/team/gender. Order by gender (M first), then rider name.

**getTeamBids(teamId: number, leagueId: number):**
Returns all transfer bids for a team in a league, joined with `riders` table twice (aliased as `outRider` and `inRider`) to get rider names. Use `alias(riders, "outRider")` and `alias(riders, "inRider")` from `drizzle-orm`. Order by `submittedAt` DESC.

Return type should include: bidId, outRiderId, outRiderName, inRiderId, inRiderName, status, reason, adminNote, submittedAt, resolvedAt.

**getActiveTransferWindow(leagueId: number):**
Returns the currently active transfer window (where `now() BETWEEN opensAt AND closesAt`). Returns null if no active window. Use `lte(transferWindows.opensAt, now)` and `gt(transferWindows.closesAt, now)`.

**getTeamTransferCount(teamId: number, leagueId: number, windowId: number):**
Counts approved transfers for a team within a specific transfer window's time range. Used for limit checking. Query `transferBids` where `status = 'approved'` AND `submittedAt` falls within the window's `opensAt`/`closesAt` range.

Export all functions and their return types (use `Awaited<ReturnType<typeof fn>>[number]` or explicit types).

Import from: `@/lib/db`, `@/db/schema/transfers`, `@/db/schema/draft`, `@/db/schema/riders`, `@/db/schema/leagues`. Use `drizzle-orm` operators: `eq`, `and`, `notInArray`, `lte`, `gt`, `gte`, `desc`, `count`, `alias`.

**Server Actions (`actions.ts`):**

`submitTransferBid(formData: { leagueId, outRiderId, inRiderId, reason? })`:
1. Auth: `getAuthenticatedUser()` from `league-auth.ts`, then `checkLeagueMembership`
2. Zod validation with schema: `{ leagueId: z.number(), outRiderId: z.number(), inRiderId: z.number(), reason: z.string().optional() }`
3. League status guard: fetch league, check `status === "active"`
4. Validate outRider belongs to this team (query `draftPicks` for `teamId + leagueId + outRiderId`)
5. Validate inRider is a free agent (no `draftPick` exists for `leagueId + inRiderId`)
6. **Gender constraint (user decision #5):** Fetch both riders from `riders` table. Verify `outRider.gender === inRider.gender`. If not, return error: "You can only swap riders within the same gender pool (men for men, women for women)"
7. **Transfer window validation (user decision #4):** Call `getActiveTransferWindow(leagueId)`. If no window, return error: "No active transfer window". If window has `maxTransfers`, call `getTeamTransferCount` and check against limit
8. Insert into `transferBids` with status "pending"
9. Insert into `transferAudit` with action "SUBMITTED"
10. `revalidatePath` both `/leagues/${leagueId}/transfers` and `/admin/transfers`
11. Return `{ success: true }`

`cancelTransferBid(bidId: number, leagueId: number)`:
1. Auth + membership check
2. Fetch bid, verify it belongs to user's team AND status is "pending"
3. Update bid status to "cancelled", set resolvedAt
4. Insert audit entry with action "CANCELLED"
5. Revalidate both paths
6. Return `{ success: true }`
