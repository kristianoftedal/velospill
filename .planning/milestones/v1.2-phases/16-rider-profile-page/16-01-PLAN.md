Create src/lib/rider-queries.ts implementing getRiderSeasonProfile(riderId: number). The function fetches all data for a rider profile in three separate queries (do not try to combine into one massive join — keep it readable):

**Query 1 — Rider metadata + total points:**
```ts
const riderRow = await db
  .select({
    id: riders.id,
    name: riders.name,
    team: riders.team,
    nationality: riders.nationality,
    gender: riders.gender,
    totalPoints: sql<number>`COALESCE(SUM(${raceResults.points}), 0)`,
  })
  .from(riders)
  .leftJoin(raceResults, eq(raceResults.riderId, riders.id))
  .where(eq(riders.id, riderId))
  .groupBy(riders.id, riders.name, riders.team, riders.nationality, riders.gender)
  .limit(1)
```
Return null if no row found (rider does not exist).

**Query 2 — Per-race scoring breakdown with categories:**
Get all race results for this rider, joined to races. Group by raceId so each race has one row with an array of category scores. Use a raw SQL aggregate or do it in application code:
```ts
const resultsRows = await db
  .select({
    raceId: races.id,
    raceName: races.name,
    raceType: races.raceType,
    startDate: races.startDate,
    parentRaceId: races.parentRaceId,
    category: raceResults.category,
    position: raceResults.position,
    points: raceResults.points,
  })
  .from(raceResults)
  .innerJoin(races, eq(races.id, raceResults.raceId))
  .where(eq(raceResults.riderId, riderId))
  .orderBy(asc(races.startDate), asc(raceResults.category))
```
Then group in application code by raceId, aggregating category scores into an array. For each race, compute racePoints = sum of all category points for that race.

**Query 3 — Ownership history:**
Determine which team held the rider at each race using draftPicks. The ownership-at-race-time pattern (established in Phase 6): the team that owned the rider at a race is the draftPick where pickedAt <= race.startDate AND (no later draftPick for the same rider in the same league exists with pickedAt <= race.startDate).

Use a simpler approach: fetch all draftPicks for this rider across all leagues with team name, ordered by pickedAt ASC. Then for each race in resultsRows, find the active ownership entry = the draftPick with the most recent pickedAt that is still <= race.startDate.

```ts
const ownershipRows = await db
  .select({
    leagueId: draftPicks.leagueId,
    teamId: teams.id,
    teamName: teams.name,
    pickedAt: draftPicks.pickedAt,
  })
  .from(draftPicks)
  .innerJoin(teams, eq(teams.id, draftPicks.teamId))
  .where(eq(draftPicks.riderId, riderId))
  .orderBy(asc(draftPicks.pickedAt))
```

Build ownership history per race: for each RiderRaceEntry, find ownershipRows where pickedAt <= race.startDate. Among those, the one with the latest pickedAt is the active owner. If multiple leagues owned the rider, include all league-team pairs (a rider can be on different teams in different leagues simultaneously).

Build RiderOwnershipEntry[] — one entry per (raceId, leagueId) pair:
